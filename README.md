# JavaScript PlayWright Projects

Collection of JavaScript practice files and Playwright examples organized by chapter.

## Overview
- This repository contains chapter-organized JavaScript examples, practice problems, and Playwright automation snippets.
- The `Practice_JS_Programs` folder is intentionally excluded from the repository (listed in `.gitignore`).

## Structure (top-level folders)
- `Chapter_1_JSBasics/` — Basic JS examples and exercises.
- `CHapter_2_JSBasics/` — Additional basics and practice files.
- `Chapter11_Arrays/` — Array-related examples and exercises.
- `Chapter12_Functions/` — Function-related examples.
- `Chapter13_Strings/` — String methods and examples.
- `Chapter14_Objects/` — Object patterns and methods.
- `Chapter15_2D_Array/` — 2D array examples.
- `Chapter16_Callback/` and `Chapter17_Promise/` — Async patterns and callbacks.
- `Chapter4_Javascript_Concepts/` — JS concepts and examples.

## Requirements
- Node.js (v14+) to run example scripts.

## Running examples
From the repository root, run a script with Node, for example:

```bash
node Chapter_1_JSBasics/hello.js
```
