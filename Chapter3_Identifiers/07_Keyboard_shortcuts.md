# VS Code Windows Keyboard Shortcuts (Windows)

| Action | Shortcut |
| --- | --- |
| New Window | Ctrl+Shift+N |
| Close Window | Ctrl+Shift+W |
| Close Window (OS) | Alt+F4 |
| Toggle Full Screen | F11 |
| Toggle Zen Mode | Ctrl+K Z |
| Toggle Side Bar Visibility | Ctrl+B |
| Split Editor | Ctrl+\ |
| Focus 1st Editor Group | Ctrl+1 |
| Focus 2nd Editor Group | Ctrl+2 |
| Focus 3rd Editor Group | Ctrl+3 |
| Close Editor Group | Ctrl+K W |
| Toggle Panel | Ctrl+` |
| Toggle Maximized Editor | Ctrl+K Ctrl+\ |
| Cycle Through Open Windows | Alt+Tab |
| Open Recent Window | Ctrl+R |
